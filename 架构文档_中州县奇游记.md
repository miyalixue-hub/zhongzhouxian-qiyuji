## 一、项目概述

**核心功能：** 儿童通过6步互动选择（选神兽→贴纹饰→选表情→选颜色→选装饰→确认提示词），由AI生成手绘风格的怪兽图片，再一键转化为3D模型，最终可3D打印成实体手办。

### 3.2 AI图片生成（2D）

| 属性 | 说明 |
|------|------|
| **服务商** | 火山引擎（字节跳动） |
| **API端点** | `https://ark.cn-beijing.volces.com/api/v3/images/generations` |
| **模型** | `doubao-seedream-4-5-251128`（Seedream 4.5） |
| **认证方式** | Bearer Token（API Key存在localStorage） |
| **调用方式** | 浏览器直接调用（无中间代理） |
| **生成数量** | 每次生成4张候选图 |
| **分辨率** | 1024×1024 |

**数据流：**
```
用户选择参数 → 构建prompt → 调用火山引擎API → 返回4张图片URL
→ 浏览器下载图片并转为base64存储 → 用户在page-10选择1张
→ 将选中图片的base64数据传给Meshy生成3D
```

**关键设计：**
- 图片生成后立即转为base64存储，避免火山引擎临时URL过期导致后续3D生成失败
- API Key通过页面设置面板输入，存储在localStorage（`ark_api_key`键名）

### 3.3 3D模型生成（Meshy）

| 属性 | 说明 |
|------|------|
| **服务商** | Meshy.ai |
| **API端点** | `https://api.meshy.ai/openapi/v1/image-to-3d` |
| **模型** | `latest`（即Meshy 6） |
| **认证方式** | Bearer Token（通过Worker代理头`X-Meshy-Key`传递） |
| **输出格式** | GLB + STL + 3MF（显式请求三种格式） |
| **模型参数** | 三角面片、目标3万面、生成纹理+PBR贴图 |

**异步工作流（轮询模式）：**
```
1. POST /api/image-to-3d  →  返回 task_id
2. 每3秒 GET /api/image-to-3d/{task_id}  →  查询进度
3. 状态流转：PENDING → IN_PROGRESS → SUCCEEDED/FAILED
4. 成功时返回：model_urls（含glb/stl/3mf链接）+ thumbnail_url
5. 最长等待5分钟（maxPollTime: 300000ms）
```

**为什么需要Cloudflare Worker代理？**
- Meshy API不支持CORS，浏览器无法直接调用
- Worker负责：添加CORS头 → 转发请求 → 将Meshy API Key注入请求头
- 用户的API Key从浏览器通过`X-Meshy-Key`自定义头传给Worker

**关键设计：**
- 生成完成后立即显示thumbnail预览图（秒出），model-viewer在后台加载3D模型
- 3D模型加载成功后自动切换到3D交互视图；如果加载超时/失败，保留预览图作为降级方案
- 下载按钮支持：3MF（优先）+ GLB + 打印工单PDF

## 六、安全与成本分析

### 6.1 安全风险

1. **API Key泄露风险**：
   - 火山引擎API Key和Meshy API Key通过设置面板输入，存储在浏览器localStorage
   - 任何XSS漏洞或恶意脚本都可读取这些Key
   - Worker代理的CORS全开放，理论上任何人都可以向Worker发送请求

2. **Meshy积分滥用风险**：
   - 每次3D生成消耗约30积分（Pro会员每月2000积分）
   - 如果代理URL泄露，他人可消耗你的积分

3. **火山引擎费用风险**：
   - Seedream 4.5每次生成4张图的费用（按量计费）
   - API Key如果泄露可能被大量调用

### 6.2 成本控制

| 服务 | 计费方式 | 当前用量 | 备注 |
|------|----------|----------|------|
| 火山引擎 | 按调用次数 | 每次4张图 | 研学场景预计每次课程消耗有限 |
| Meshy | 积分制 | ~30积分/次3D | Pro会员月2000积分≈66次3D生成 |
| GitHub Pages | 免费 | 无成本 | 有带宽限制但对低流量足够 |
| Cloudflare Workers | 免费额度 | 10万次/天 | 远超研学使用量 |
