# 中轴奇游记 · 首页改造方案

## 一、改造目标

将首页从"功能堆砌页"改造为"一张沉浸式的北京中轴线旅程地图"。

**核心原则：**
- 做减法：去掉冗余的交互元素，只保留"地图 + 9站故事"
- 做统一：所有站点卡片使用同一套卷轴视觉语言
- 做故事：每站都有钩子文案，让用户想解锁

## 三、砍掉的元素

| 原有元素 | 状态 | 理由 |
|---------|------|------|
| 蛇形铁轨SVG | ❌ 去掉 | 和"中轴线"的直线概念矛盾，信息密度低 |
| 铛铛车emoji + 铁轨动画 | ❌ 去掉 | 首页不需要车在跑，旅程感靠站点顺序传达 |
| 旅程对话气泡（journey-dialogue-area） | ❌ 去掉 | 和站点卡片功能重叠 |
| NPC铛铛车引导气泡 | ❌ 去掉 | 序章已有人设，不需要重复 |
| 正阳门"英雄卡"大尺寸独占布局 | ⚠️ 降级 | 保留为和其他站同规格的卡片，但标记"已解锁" |
| 底部Toast | ❌ 去掉 | 锁定站点直接在卡片上体现锁定状态 |
| 底部进度条（独立版） | ⚠️ 合并 | 进度信息合并到序章卷轴内 |

**卡片视觉规范：**
- 背景：宣纸渐变（`#fdf8ef` → `#f5e8c8`）
- 边框：金色双线（`rgba(184,148,62,0.25)`）
- 圆角：不规则卷轴角（`15px 4px 10px 18px` 等）
- 插图区：160x160 方块，`object-fit: contain`，圆角8px
- 站名：楷体 24px，朱砂色（已解锁）或墨色（锁定）
- 故事钩子：楷体 14px，行高 1.8，`#5a4a3a`
- 状态标签：卷轴小标（已解锁=朱砂红底，锁定=灰色）
- 操作按钮：已解锁显示"进入→"，锁定显示灰色不可点

**已解锁 vs 锁定 的差异：**

| 状态 | 卡片表现 |
|------|---------|
| 已解锁（永定门） | 正常色彩，插图清晰，底部有"进入"按钮 |
| 已解锁可进入（正阳门） | 正常色彩 + "第一章 · 已解锁"标签，有"进入"按钮 |
| 锁定（其他7站） | 整体降饱和+轻微模糊，插图灰度，无按钮，显示锁图标 |

## 五、9站文案设计

### 设计原则
- 每站一句话（不超过40字），制造悬念，让人想解锁
- 语气：轻松、有画面感、带一点"不正经"（和正阳门的语气保持一致）
- 不解释历史，只抛出一个让人好奇的点

### 完整文案

| # | 站名 | 故事钩子 | 副标题 |
|---|------|---------|--------|
| 1 | 永定门 | 中轴线的起点。名字里藏着这座城市最朴素的愿望——永远安定。 | 南大门 · 城防之始 |
| 2 | 天坛 | 皇帝在这里和天对话。圜丘坛的石板数，藏着古人的宇宙密码。 | 祭天之坛 · 天人感应 |
| 3 | 先农坛 | 皇帝亲自下田的地方。天下最尊贵的农夫，种的是象征性的那一亩三分地。 | 亲耕之坛 · 劝课农桑 |
| 4 | 正阳门 | 这哥们叫蚣蝮，龙的第九个儿子。被忘了三百年，脾气不太好。 | 正门守护 · 蚣蝮与吸水兽 |
| 5 | 天安门 | 从皇城的门到国家的门。它改过三次名字，每一次都是时代的转折。 | 皇城正门 · 国之门 |
| 6 | 故宫 | 72万平方米的迷宫。9999间半房子，多出来的那半间到底在哪？ | 紫禁之巅 · 万国朝宗 |
| 7 | 景山 | 爬上这座人工山，你能看到中轴线的全貌——也能看到一座城的命运。 | 万岁山 · 登高远眺 |
| 8 | 万宁桥 | 桥下的镇水兽，已经趴了七百年。它见过元朝的船，也见过明朝的雨。 | 中轴第一桥 · 七百年守望 |
| 9 | 钟鼓楼 | 没有钟表的年代，这两座楼就是全北京人的闹钟。晨钟暮鼓，一天从这里开始。 | 时间之楼 · 晨钟暮鼓 |

正阳门保留现有的完整故事文案（蚣蝮传说），不缩减。但卡片尺寸和其他站一致，不搞独占式英雄卡。

## 六、中轴线的视觉处理

### 方案：直线贯穿（非蛇形铁轨）

```
        │
   ┌────┴────┐
   │ 永定门  │
   └────┬────┘
        │
   ┌────┴────┐
   │  天坛   │
   └────┬────┘
        │
       ...
        │
   ┌────┴────┐
   │ 钟鼓楼  │
   └─────────┘
```

- 中轴线用**毛笔飞白风格直线**（保留现有 `axis-line` 样式）
- 站点卡片**左右交替排列**（奇数靠左、偶数靠右）
- 轴线上用小圆点标记每个站点位置
- 底部终点标"中轴线起点"（永定门在南端）

**为什么不用蛇形铁轨？**
- 中轴线是"直线"，蛇形违背了这个核心意象
- 直线布局让用户一眼看到全貌，符合"一条线串联半部北京城"
- 直线更容易在手机端适配

### 移动端适配（≤768px）

```
        │
   ┌────┴────────────────┐
   │ [插图] 永定门        │
   │ 故事钩子...          │
   └─────────────────────┘
        │
   ┌────┴────────────────┐
   │ [插图] 天坛          │
   │ 故事钩子...          │
   └─────────────────────┘
        │
       ...
```

移动端改为全宽卡片，中轴线缩为左侧装饰线。

## 七、交互流程

### 进入首页
1. 看到序章卷轴 + 9站地图
2. 正阳门卡片高亮（已解锁），其余灰度
3. 用户点击"进入正阳门" → 跳转到正阳门研学流程

### 解锁逻辑（未来扩展）
- 当用户完成正阳门第一章后，天坛/永定门自动解锁
- 解锁动画：卡片从灰度渐变为彩色，锁图标消失
- 解锁后卡片出现"进入→"按钮

### 点击锁定站点
- 不做任何跳转，卡片轻微弹跳 + 显示"即将开放，敬请期待！"（Toast 3秒消失）
