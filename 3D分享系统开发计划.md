# 中轴奇游记 - 3D模型分享系统开发计划

## 目标
AI魔法师训练营生成的3D模型 → 自动修复为可打印状态 → 生成H5分享页 → 孩子一键分享到微信给家长

## 系统架构

```
┌─────────────────────┐        ┌───────────────────────────┐
│  前端 (Cloudflare)    │        │  后端 (腾讯云 SCF)         │
│  mindbubble.cloud    │  API   │                           │
│                     │ ─────→ │  ① 接收STL + 自动修复      │
│  - 现有研学页面        │        │  ② 存COS + 生成分享链接    │
│  - 新增"分享"按钮      │        │  ③ 提供分享页数据接口      │
└─────────────────────┘        └───────────────────────────┘
                                        │
                                        ↓
                               ┌──────────────────┐
                               │  腾讯COS对象存储   │
                               │                   │
                               │  - STL文件(修复后)  │
                               │  - 2D图片          │
                               │  - 作品元数据       │
                               └──────────────────┘
                                        │
                                        ↓
                               ┌──────────────────┐
                               │  H5 分享页         │
                               │  share.mindbubble │
                               │  .cloud           │
                               │                   │
                               │  - 2D作品展示      │
                               │  - 3D模型旋转预览   │
                               │  - STL下载按钮     │
                               │  - 微信分享卡片    │
                               └──────────────────┘
```

## 模块拆分

### 模块一：STL 自动修复脚本
- **做什么**：Python 脚本，Meshy 返回 STL 后自动修复（切平底座、去碎片、修网格）
- **技术**：trimesh + numpy-stl
- **部署**：先本地可用，后续接入腾讯云函数

### 模块二：腾讯云基础设施
- **COS 对象存储**：存 STL 文件 + 2D 图片
- **SCF 云函数**：接收上传、触发修复、生成分享链接
- **API 网关**：前端通过 HTTPS 调用
- **自定义域名**：API 走 share.mindbubble.cloud 或类似子域名

### 模块三：H5 分享页
- **做什么**：一个漂亮的移动端适配页面，展示孩子作品
- **功能**：
  - 展示 2D 神兽图片
  - Three.js 3D 模型旋转预览（自动旋转，手指可拖拽）
  - "下载 STL" 按钮（家长下载后去打印）
  - 微信分享卡片（标题+缩略图+描述）
  - 底部品牌："AI 魔法师训练营 · 中轴奇游记"

### 模块四：前端集成
- 现有研学页面增加"分享作品"流程
- 生成完成 → 自动上传到后端 → 生成分享链接 → 展示分享按钮
- Meshy API Key 迁移到后端（安全）

## 技术选型

| 组件 | 选型 | 理由 |
|------|------|------|
| 后端 | 腾讯云 SCF 云函数 | 按调用计费，小项目免费 |
| 存储 | 腾讯云 COS | 与 SCF 同区域，速度快 |
| 3D预览 | Three.js | 轻量，微信内置浏览器兼容好 |
| H5页面 | 纯 HTML/CSS/JS | 与现有项目技术栈一致 |
| STL修复 | trimesh (Python) | 最成熟的3D网格处理库 |
